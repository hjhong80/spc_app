## 참고 메모
- 시리얼 검색 기능은 최종적으로 "세부 일 차트에서 점을 직접 클릭했을 때와 동일한 상세 모달 결과"를 기준 동작으로 맞춘다.
- 검색 진입점은 차트 유형과 무관하게 공용으로 두고, 상세 조회는 세부 일 차트 컨텍스트를 기준으로 처리하는 방향이 단순하다.
- CPK 역산은 단순 조회가 아니라 조합 탐색 문제에 가깝다. SQL만으로 풀기보다 후보군 조회는 SQL, CPK 계산과 최적 조합 선택은 Java 서비스에서 처리하는 방향이 현실적이다.
- 목표 CPK 정확 일치가 필수가 아니므로, 탐색 결과가 기준치 이상이면 반환하고 기준치 미달이면 검색 실패로 처리하는 정책이 적합하다.
