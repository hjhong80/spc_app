# 데이터 조회 패널 레이아웃 정리 설계

## 목표
- 데이터 조회 패널의 상단 위계를 단순화한다.
- 불필요한 설명 문구를 제거한다.
- `프로젝트 검색 + 새로고침` 영역을 `프로젝트 리스트` 섹션 아래로 이동해 리스트 탐색 흐름을 정리한다.

## 변경 방향
- `데이터 조회` 제목 아래의 `프로젝트를 선택해 저장된 데이터를 조회합니다.` 문구를 제거한다.
- 상단 구조를 `데이터 조회` -> `프로젝트 리스트` -> `프로젝트를 클릭하면 차트 보기 화면으로 이동합니다.` -> `프로젝트 검색 + 새로고침` -> 리스트 박스 순서로 재배치한다.
- 기존 검색/새로고침 동작, 리스트 선택 동작, 상태 메시지 동작은 유지한다.

## 검증 포인트
- 설명 문구가 더 이상 렌더링되지 않는지 확인한다.
- `프로젝트 리스트` 텍스트와 안내 문구가 검색 입력창보다 앞에 렌더링되는지 확인한다.
- 기존 프로젝트 목록 렌더링과 빈 상태 메시지가 그대로 유지되는지 확인한다.
