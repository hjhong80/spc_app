# 데이터 조회 하단 검색 툴바 설계

## 목표
- 데이터 조회 패널에서 검색 입력창과 새로고침 버튼을 리스트 박스 아래로 이동한다.
- 리스트 영역을 먼저 보여주고, 검색/새로고침은 하단 제어 바처럼 배치해 시선 흐름을 단순화한다.

## 변경 방향
- 상단 구조는 `데이터 조회 -> 프로젝트 리스트 -> 안내 문구 -> 리스트 박스` 순서를 유지한다.
- `프로젝트 검색 + 새로고침` 영역은 리스트 박스 아래, 상태 메시지 위로 이동한다.
- 검색, 새로고침, 프로젝트 선택 동작은 그대로 유지한다.

## 검증 포인트
- 검색 입력창이 더 이상 리스트 안내 문구 바로 아래에 오지 않는지 확인한다.
- 빈 상태 또는 목록 상태에서 리스트 영역이 검색 입력보다 먼저 렌더링되는지 확인한다.
- 기존 제목, 안내 문구, 상태 메시지 렌더링은 유지되는지 확인한다.
