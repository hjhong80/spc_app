# 차트 메시지 컨텍스트 리셋 설계

## 목표
- 월차트 내부의 `캔들 <-> 정규분포` 전환에서는 메시지 히스토리를 유지한다.
- 월차트에서 일차트로 들어가거나 전체 차트로 돌아갈 때 메시지 히스토리를 초기화한다.

## 변경 방향
- 차트 화면 상태를 `main`, `detail-month`, `detail-day` 세 가지 컨텍스트 키로 계산한다.
- 메시지 히스토리와 활성 알림은 이 컨텍스트 키가 바뀔 때만 리셋한다.
- `detailVisualMode`는 컨텍스트 키에 포함하지 않는다.

## 검증 포인트
- `detail-month -> detail-day` 전환 시 리셋된다.
- `detail-month -> main` 전환 시 리셋된다.
- `detail-month` 내부에서 `캔들 <-> 정규분포` 전환은 유지된다.
