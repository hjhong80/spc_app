# 차트 성능칩 스낵바 전환 설계

## 목표
- 월차트/일차트 우측 상단의 성능 칩을 제거한다.
- 성능 요약은 2초 동안만 보이는 정보 스낵바로 대체한다.
- 차트 페이지의 스낵바 메시지가 누적될 때 로그아웃 오른쪽 메시지 아이콘 배지에 개수를 표시한다.

## 변경 방향
- `Chart.jsx`에서 정규분포 성능 요약 `Chip` 렌더링을 제거한다.
- 성능 요약은 메시지 히스토리에 `info` 알림으로 추가하고, 자동으로 2초짜리 스낵바를 띄운다.
- 성공, 경고, 노이즈 경고도 동일한 메시지 히스토리에 누적한다.
- 헤더 우측에는 `MessageIcon + Badge`를 두고 누적 메시지 수를 표시한다.
- 아이콘 클릭 시 팝오버에서 누적된 알림 목록을 다시 볼 수 있게 한다.

## 검증 포인트
- 성능 요약 `Chip`이 더 이상 렌더링되지 않는지 확인한다.
- 성능 메시지의 자동 숨김 시간이 2초인지 확인한다.
- 메시지 히스토리 개수가 배지 숫자로 계산되는지 확인한다.
- 심각도 우선순위가 `error > warning > success > info`로 동작하는지 확인한다.
