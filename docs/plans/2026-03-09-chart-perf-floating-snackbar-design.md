# 차트 성능 알림 위치 조정 설계

## 목표
- 성능 알림이 토글 버튼을 덮지 않게 한다.
- 성능 알림이 토글 버튼 바로 아래에 뜨도록 바꾼다.
- 성능 알림 표면을 반투명하게 조정한다.

## 변경 방향
- 성능 알림은 전역 `Snackbar`가 아니라 토글 버튼 그룹을 기준으로 한 `Popper`로 렌더링한다.
- 앵커는 세부 차트의 `ToggleButtonGroup` 래퍼를 사용한다.
- 성능 알림의 배경은 `rgba(...)` 기반 반투명 색으로 통일한다.
- 기존 성공/경고 메시지 히스토리 및 배지 개수는 유지한다.

## 검증 포인트
- 성능 알림이 `info`일 때만 토글 버튼 아래의 floating layer로 렌더링되는지 확인한다.
- 성능 알림 표면 스타일이 반투명 값을 반환하는지 확인한다.
- 일반 메시지의 히스토리, 배지 개수, 팝오버는 유지되는지 확인한다.
