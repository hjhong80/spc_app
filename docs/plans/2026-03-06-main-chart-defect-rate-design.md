# 메인 차트 불량률 막대 그래프 설계

## 작업 목적
- 메인 SPC 차트를 `평균 ± 2σ / min-max` 캔들 차트에서 `Cpk 기반 불량률` 막대 그래프로 전환한다.
- `Cpk -> 시그마 수준 -> 불량률` 계산을 백엔드에 모아 계산식 일관성을 확보한다.
- 불량률 축을 고정 8눈금으로 제공해 차트 해석성을 높인다.

## 현재 구조
- 백엔드 `chart-stats` API는 특성치별 `nominal`, `uTol`, `lTol`, `measuredMin`, `measuredMax`, `measuredMean`, `measuredSigma`, `sampleCount`를 집계해 반환한다.
- 프론트 `Chart.jsx`는 응답값으로 `USL`, `LSL`, `Cpk`를 계산하고 메인 캔들 차트를 그린다.
- 메인 차트와 세부 차트 모두 ApexCharts를 사용한다.

## 변경 설계

### 1. 계산 책임 분리
- 백엔드에서 `USL = nominal + uTol`, `LSL = nominal + lTol`를 계산한다.
- 백엔드에서 `Cpu`, `Cpl`, `Cpk`, `sigmaLevel = 3 * Cpk`, `goodRatePercent`, `defectRatePercent`를 계산한다.
- `goodRatePercent`와 `defectRatePercent`는 정규분포 누적분포함수 기반으로 계산한다.
- 계산 불능 값은 `null`로 유지한다.

### 2. 안전 처리
- 시그마 수준 변환은 `sigmaLevel = 3 * Cpk`만 사용한다.
- `Cpk >= 2.0`은 6시그마 이상으로 보고 차트상 최상단 기준선에 클램프한다.
- `Cpk <= 0` 또는 계산 불능은 최하단 불량률 100% 기준으로 클램프한다.
- 불량률은 최종적으로 `0% ~ 100%` 범위로 클램프한다.

### 3. 메인 차트 표현
- 메인 차트는 단일 막대 시리즈로 변경한다.
- 각 막대는 특성치 하나를 의미하며, 높이는 `불량률을 축 좌표로 변환한 값`이다.
- 세부 차트 진입 클릭 동작은 유지한다.

### 4. Y축 규칙
- Y축은 총 8개 기준 구간을 사용한다.
- `-1`: 하단 시인성 확보용 여백
- `0`: 불량률 0 / 6시그마 이상
- `1`: 6시그마
- `2`: 5시그마
- `3`: 4시그마
- `4`: 3시그마
- `5`: 2시그마
- `6`: 1시그마
- `7`: 불량률 100%
- 실제 막대 높이는 `불량률`을 `시그마 눈금 좌표`로 변환한 연속값을 사용한다.

### 5. 툴팁/라벨
- 메인 차트 툴팁은 기존 기본 메타 정보에 아래 필드를 추가한다.
- `Cpk`
- `시그마 수준`
- `불량률(%)`
- `양품율(%)`
- 기존 `σ` 표기는 표준편차와 혼동되므로, 메인 차트 툴팁에서는 `표준편차`와 `시그마 수준`을 명시적으로 분리한다.

## 주요 변경 사항
- 백엔드 DTO 확장: `ChartStatRespDto`에 SPC 해석용 파생 필드 추가
- 백엔드 서비스 로직 추가: Cpk/시그마/불량률 계산 helper 도입
- 프론트 API 모델 재생성
- 프론트 메인 차트 시리즈를 캔들에서 막대 차트로 전환
- 프론트 메인 Y축을 불량률/시그마 전용 고정 축으로 교체
