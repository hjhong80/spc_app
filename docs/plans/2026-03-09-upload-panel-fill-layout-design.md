# 업로드 패널 하단 정렬 설계

## 목표
- 드래그앤드롭존이 업로드 패널의 남는 세로 공간을 채우게 한다.
- `LOT 번호` 입력창과 업로드 버튼을 패널 하단 영역으로 밀어낸다.
- 결과적으로 업로드 버튼 하단이 성적서 등록 박스의 하단 패딩선 근처에 정렬되도록 한다.

## 변경 방향
- 업로드 패널을 `상단 정보 영역`, `가변 드롭존 영역`, `하단 컨트롤 영역`으로 분리한다.
- 드롭존은 `flex: 1`로 남는 높이를 차지한다.
- `LOT 번호`와 업로드 버튼은 하단 그룹으로 묶고 패널 바닥 쪽으로 유지한다.

## 검증 포인트
- 드롭존 섹션이 가변 영역으로 렌더링되는지 확인한다.
- 하단 컨트롤 그룹이 별도 래퍼로 분리되어 드롭존 아래에 유지되는지 확인한다.
- 기존 업로드 진행률 버튼과 알림 Popper 동작은 유지되는지 확인한다.
